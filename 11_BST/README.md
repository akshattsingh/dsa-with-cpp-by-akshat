# Binary Search Tree (BST) in C++

This folder contains implementation of Binary Search Tree concepts from basic to advanced level using C++.

A Binary Search Tree is a special type of Binary Tree where:
- Left subtree contains values smaller than root
- Right subtree contains values greater than root
- Both subtrees are also BSTs

## Topics Covered

### 1️⃣ Basic Operations
- BST Creation
- Insert into BST
- Delete from BST
- Search in BST

### 2️⃣ Validation & Properties
- Validate BST
- Inorder Traversal (Sorted Output)
- Time Complexity Analysis

### 3️⃣ Important Interview Problems
- Lowest Common Ancestor (LCA) in BST
- Kth Smallest Element
- Inorder Successor
- Sorted Array to Balanced BST 

## Why BST is Important?

- Very common in coding interviews
- Foundation for AVL Tree & Red-Black Tree
- Used in database indexing
- Helps understand ordered data structures
- Inorder traversal gives sorted output
